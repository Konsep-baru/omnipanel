ğŸ³ CONTAINERS (11 perintah)

```
container ls              - Lihat container yang sedang running
container ls -a           - Lihat semua container (termasuk yang sudah stop)
container run <image>     - Jalankan container baru dari image (auto-pull)
                           Contoh: container run nginx:alpine
                           
container stop <nama>     - Stop container yang sedang running
                           Contoh: container stop web
                           
container start <nama>    - Start container yang sedang stop
                           Contoh: container start web
                           
container restart <nama>  - Restart container
                           Contoh: container restart web
                           
container rm <nama>       - Hapus container
                           Contoh: container rm web
                           
container logs <nama>     - Lihat 50 baris terakhir log container
                           Contoh: container logs web
                           
container logs <nama> -f  - Follow log container (real-time)
                           Contoh: container logs web -f
                           
container exec <nama> <cmd> - Jalankan perintah di dalam container
                           Contoh: container exec web bash
                           
container stats           - Lihat statistik resource semua container
```
